# Userscripts

Моя личная коллекция юзерскриптов для Tampermonkey, улучшающих работу с различными сайтами.

## Структура репозитория

Каждый скрипт или группа настроек для определенного сайта будет находиться здесь. Файлы скриптов должны иметь расширение `.user.js` для автоматического распознавания Tampermonkey.

## Как установить скрипт

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера.
2. Откройте нужный `.user.js` файл в этом репозитории и нажмите кнопку **Raw** — Tampermonkey сам предложит установить скрипт.
